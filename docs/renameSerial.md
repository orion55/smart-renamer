# renameSerial.ps1 — PowerShell скрипт для автоматического переименования сериалов

## Назначение

Скрипт обрабатывает папку с загруженными видеофайлами сериалов (`f:\Сериалы`), приводит их к удобному виду и переводит названия на русский язык через Yandex Translate API.

## Алгоритм работы

### 1. Инициализация

- Получает IAM-токен Yandex Cloud через OAuth для доступа к Translate API
- Создаёт лог-файл `log/serial.log`

### 2. Обработка папок с сериями (если в директории есть подпапки)

**Разворачивание вложенных папок** — если серия лежит в нескольких уровнях вложенности, все видеофайлы поднимаются наверх, лишние папки и не-видео файлы удаляются (оставляются только `.mkv`, `.avi`, `.mp4`, `.ts`)

**Одиночный видеофайл в папке** — файл перемещается в корень директории сериалов, пустая папка удаляется

**Несколько видеофайлов в папке (сезон/сериал)** — удаляются `.srt` субтитры, затем файлы переименовываются по шаблону:
- `S01E05.mkv` → `05.mkv` (первый сезон — только номер эпизода)
- `S02E05.mkv` → `0205.mkv` (второй сезон и выше — сезон+эпизод)
- Поддерживаются форматы: `SxxExx`, `.xx.`, `.Sxx.Exx.`

**Переименование папок** — из имени папки вырезается маркер сезона (`_S01_`, `.S01.`), затем точки/подчёркивания заменяются пробелами и название переводится на русский

### 3. Обработка отдельных видеофайлов

- Из имени файла вырезается год `(2024)`, `.2024.` и т.п.
- Точки и подчёркивания заменяются пробелами
- Название переводится на русский через Yandex Translate
- При конфликте имён дубликат удаляется

## Библиотеки

- **`lib.ps1`** — утилиты: `testDir`, `createDir`, `testFiles`, настройка цветов консоли
- **`PSMultiLog.ps1`** — мультиканальное логирование (файл, консоль, email, Slack, EventLog)

## Внешние зависимости

- **Yandex Cloud IAM** — авторизация через OAuth-токен
- **Yandex Translate API** (`translate.api.cloud.yandex.net`) — перевод названий на русский язык
